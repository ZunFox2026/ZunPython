Python là một ngôn ngữ lập trình cấp cao, được sử dụng rộng rãi trong nhiều lĩnh vực như phát triển web, phân tích dữ liệu, trí tuệ nhân tạo,...
Python được tạo ra vào năm 1991 bởi Guido van Rossum, và được phát hành lần đầu tiên vào năm 1991.
Một số tính năng chính của Python bao gồm:
- Đơn giản và dễ đọc
- Có thể chạy trên nhiều nền tảng khác nhau
- Hỗ trợ nhiều thư viện và framework
Ví dụ về chương trình Python đơn giản:
print(Hello, World!)
