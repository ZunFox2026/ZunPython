# Giới thiệu Python
## Giới thiệu
Python là một ngôn ngữ lập trình cấp cao, được tạo ra vào cuối những năm 1980 bởi Guido van Rossum. Python được thiết kế để dễ đọc, dễ viết và dễ học, làm cho nó trở thành một trong những ngôn ngữ lập trình phổ biến nhất trên thế giới. Python được sử dụng trong nhiều lĩnh vực khác nhau, bao gồm phát triển web, phân tích dữ liệu, trí tuệ nhân tạo, và nhiều hơn nữa.

## Lý thuyết
Python là một ngôn ngữ lập trình hướng đối tượng, có nghĩa là nó tổ chức code thành các đối tượng có thể chứa dữ liệu và chức năng. Python cũng là một ngôn ngữ lập trình động, có nghĩa là nó không cần phải biên dịch code trước khi chạy. Điều này làm cho Python trở thành một ngôn ngữ lập trình linh hoạt và dễ sử dụng. Một số khái niệm cơ bản trong Python bao gồm biến, kiểu dữ liệu, vòng lặp, và hàm. Biến là một tên được gán cho một giá trị, kiểu dữ liệu xác định loại giá trị được lưu trữ, vòng lặp cho phép code được chạy nhiều lần, và hàm là một khối code có thể được gọi nhiều lần.

## Ví dụ
Dưới đây là một ví dụ đơn giản về cách sử dụng Python:
```python
x = 5  # gán giá trị 5 cho biến x
y = 3  # gán giá trị 3 cho biến y
tong = x + y  # tính tổng của x và y
print(tong)  # in tổng ra màn hình
```
Kết quả của code trên sẽ là `8`, vì `5 + 3 = 8`. Ví dụ này minh họa cách sử dụng biến, toán tử, và hàm `print()` trong Python.

## Bài tập
Để kiểm tra hiểu biết của bạn về Python, hãy thực hiện các bài tập sau:
- Tạo một biến `name` và gán cho nó giá trị là tên của bạn.
- Tạo một biến `age` và gán cho nó giá trị là tuổi của bạn.
- Sử dụng hàm `print()` để in ra màn hình các giá trị của `name` và `age`.
- Tạo một hàm `chao` có thể in ra màn hình "Xin chào, [tên của bạn]!" khi được gọi.
- Gọi hàm `chao` để kiểm tra xem nó có hoạt động đúng không.
