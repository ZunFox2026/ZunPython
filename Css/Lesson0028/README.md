# Tìm hiểu về thuộc tính Text
## Giới thiệu
Thuộc tính Text trong CSS là một phần quan trọng giúp chúng ta điều chỉnh và tùy chỉnh cách hiển thị văn bản trên trang web. Với thuộc tính này, bạn có thể thay đổi màu sắc, kích thước, phong cách và nhiều thuộc tính khác của văn bản. Trong bài viết này, chúng ta sẽ tìm hiểu về các thuộc tính Text cơ bản và cách sử dụng chúng trong CSS.

## Lý thuyết
Có nhiều thuộc tính Text trong CSS, bao gồm:
- `color`: Điều chỉnh màu sắc của văn bản.
- `font-size`: Điều chỉnh kích thước của văn bản.
- `font-family`: Điều chỉnh loại font của văn bản.
- `font-weight`: Điều chỉnh độ đậm, nhạt của văn bản.
- `text-align`: Điều chỉnh vị trí của văn bản (trái, phải, giữa).
- `text-decoration`: Điều chỉnh kiểu trang trí của văn bản (gạch chân, gạch ngang, v.v.).

Ví dụ về sử dụng thuộc tính Text trong CSS:
```css
p {
  color: blue; /* Màu xanh */
  font-size: 20px; /* Kích thước 20 pixel */
  font-family: Arial; /* Loại font Arial */
  font-weight: bold; /* Đậm */
  text-align: center; /* Vị trí giữa */
  text-decoration: underline; /* Gạch chân */
}
```

## Ví dụ
Để minh họa cách sử dụng thuộc tính Text, hãy xem xét ví dụ sau:
Giả sử chúng ta có một đoạn văn bản và muốn áp dụng các thuộc tính Text như sau:
- Màu sắc: Xanh lá cây
- Kích thước: 18 pixel
- Loại font: Times New Roman
- Độ đậm: Nhạt
- Vị trí: Trái
- Kiểu trang trí: Gạch ngang

CSS sẽ như sau:
```css
.example {
  color: green; /* Màu xanh lá cây */
  font-size: 18px; /* Kích thước 18 pixel */
  font-family: 'Times New Roman'; /* Loại font Times New Roman */
  font-weight: normal; /* Độ đậm nhạt */
  text-align: left; /* Vị trí trái */
  text-decoration: line-through; /* Gạch ngang */
}
```

## Bài tập
Để thực hành và nắm vững thuộc tính Text, bạn có thể thử các bài tập sau:
- Tạo một đoạn văn bản và áp dụng các thuộc tính Text khác nhau (màu sắc, kích thước, loại font, độ đậm, vị trí, kiểu trang trí).
- Thử thay đổi các giá trị của thuộc tính Text và quan sát cách chúng ảnh hưởng đến văn bản.
- Sử dụng các thuộc tính Text để tạo ra các hiệu ứng khác nhau trên trang web, chẳng hạn như tạo tiêu đề, chú thích, hoặc các phần văn bản đặc biệt.
